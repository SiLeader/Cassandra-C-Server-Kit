# cqlsvrkit - Cassandra CQL Server side RPC system for C++

&copy; 2020 SiLeader and Cerussite.

## features
+ CQL server RPC
+ multi-threading support

## dependencies
+ C++14
+ Boost C++ Libraries
  + Boost.Asio
  + Boost.Optional
  + Boost.Variant
  + `boost::string_view`
+ libcpp17 (submodule)
  + `cpp17::span<T>`
